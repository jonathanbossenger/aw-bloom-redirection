Load-in settings

  • Intro Animation

    Define the animation that is used, when you load the page.

  • Trigger After Time Delay

  • Delay (in seconds)

    Define how many seconds you want to wait before the pop up appears on the screen.

  • Trigger After Inactivity

  • Idle Timeout ( in seconds )

    Define how many seconds user should be inactive before the pop up appears on screen.

  • Trigger At The Bottom of Post

  • Trigger After Commenting

  • Trigger After Scrolling

  • Percentage Down The Page

    Define the % of the page to be scrolled before the pop up appears on the screen.

  • Trigger After Purchasing

    Display on "Thank you" page of WooCommerce after purchase

  • Trigger On Click

  • CSS Selector (string)

    A CSS selector string for a trigger element of your choosing. This opt-in will be triggered by clicking the element. Example: a.trigger_popup

  • Display Once Per Session

  • Session Duration (in days)

    Define the length of time (in days) that a session lasts for. For example, if you input 2 a user will only see a popup on your site every two days.
